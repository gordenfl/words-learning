# Basic Requirements

## 1. User can add a word to the list of words they want to learn.

- [ ] Take a picture from a book and analysis all the words in that picture.
- [ ] Add a new word to the list. From take picture from a book and read it from an image or by typing in the text box.
- [ ] The app can ignore all the words this user has already stored in the database. show the all the unknown words list in that picture user unknown.
- [ ] User can add the words in to the list of words with the status of unknown.
- [ ] Generate a short article with the words I know and new words I marked as unknown.
- [ ] There are a function to show the list of words I have learned, and the list of words I still need to learn.
- [ ] After user read the article, user thinks the unknown words is known, then click on the button to change the status of that word from unknown to known.
- [ ] Show the list of words I have learned, and the list of words I still need to learn.
- [ ] If there are no more words left for me to learn, generate another article with new words I don't know yet.
- [ ] Achievement System, show how many words you have learned today, week, month, year.
- [ ] Achievements: Learn 50 words per day, Learn 200 words per week, Learn 800 words per month, Learn 3650 words per year.
- [ ] Reward system, if you reach one achievement, get a reward.
- [ ] Rewards: Get a sticker, get a badge, get a certificate, get a gift card.
- [ ] Save the progress of learning words into a file or database.
- [ ] Load the progress of learning words from a file or database.
- [ ] Share the progress of learning words with friends via email, social media, etc.
- [ ] Set up reminders to review old words before forgetting them again.
- [ ] Create flashcards for each word so users can practice memorizing them through repetition and visualization techniques such as mnemonics or acronyms.
