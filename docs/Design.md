# 系统设计

想法： 我认为首先系统不应该以拍照读取汉字然后接入接入学习为主要的目标。 
    人要学习必定是从见到到复杂， 这就是为什么我在做生成文章的时候感觉很难的原因，我事先没有想好如何去制作一个整体。仅仅是通过一个图片生成文字的功能去做一个系统， 势必会带来很多问题。

## 基础的学习框架

- 每日结构（固定模板）

    * 1–2小时结构
    * 发音/听力（15分钟）
    * 新句型 + 词汇（30分钟）
    * 跟读 + 复述（30分钟）
    * 应用（说出来）（15–30分钟）

- 按照十五天进行

    * 第一周 发音 + 生存中文 ： 掌握拼音 + 声调， 会最基础表达

        - 学习： 声母， 韵母， 四声 （去 看一下Hello Chinese 怎么做的）
        - 学习简单句子：
            我是 Gordon
我是美国人
我会说一点中文
你好吗？
我很好
